Integrate in a single import.

Wrap a tool with guard() and every call it makes is evaluated before it executes. No agent rewrites. No local models.

That's it. Every tool call is now evaluated before execution.

agent.py Python 
from kelhe.adapters.langgraph import guard, AgentContext
from kelhe import Kelhe, Mode
from langchain.agents import create_agent
from langchain_openai import ChatOpenAI

kelhe = Kelhe(mode=Mode.ENFORCE, risk_threshold=7.0,
  audit_handler=splunk, api_key="kelhe...",
  escalation_handler=pagerduty)
ctx = AgentContext()
agent = create_agent(
    ChatOpenAI(model="gpt-4o-mini"),
    tools=[
        guard(kelhe, assign_roles, context=ctx)
    ]
)
ctx.bind(agent)
